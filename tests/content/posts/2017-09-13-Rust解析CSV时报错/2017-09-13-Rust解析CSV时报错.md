使用 Rust 的 csv 库时遇到了一些错误记下来以以后查询。

1.  CSV deserialize error: missing field `name`:

这个因为在将 [csv](https://docs.rs/csv/1.0.0-beta.4/csv/) 读取解析成 strcut 时， 头信息对不上造成的。
解决方法：
可以使用 [serde](https://docs.serde.rs/serde/) 库，重命名 strcut 元素对应的头：

```rust
extern crate serde;

struct Item {
    id: u32,
    #[serde(rename = "Name")]
    name: String,
}
```
